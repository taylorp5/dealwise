# DealWise Brand Assets

Logo files are included:

- **dealwise-logo.svg** - Full logo with wordmark (color, for light backgrounds)
- **dealwise-icon.svg** - Icon only (color, for favicons/app icons)
- **dealwise-logo-dark.svg** - Full logo (monochrome dark, for light backgrounds)
- **dealwise-logo-white.svg** - Full logo (monochrome white, for dark backgrounds)

The main logo (`dealwise-logo.svg`) is used in the app header. All logos use the DealWise brand gradient (blue to green) and are optimized for web use.
